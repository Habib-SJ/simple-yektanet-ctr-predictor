# simple-yektanet-ctr-predictor
CTR prediction using simple and scalable ML models

Click-Through Rate Prediction System

سیستمی ساده و قابل توسعه برای پیش‌بینی نرخ کلیک (CTR) با استفاده از مدل‌های یادگیری ماشین. شامل ابزارهای پیش‌پردازش، آموزش، و ارزیابی مدل‌ها است و می‌تواند به‌صورت API برای پیش‌بینی آنلاین استفاده شود. این پروژه یک ساختار ماژولار ارائه می‌دهد تا توسعه و آزمایش مدل‌ها به‌سادگی انجام شود.
هدف پروژه:
پیش‌بینی نرخ کلیک CTR برای تبلیغات شبیه یکتانت

تکنولوژی ها:

Python, Django, DRF
